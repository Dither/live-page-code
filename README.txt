Save snapshot of the page for Opera
=============

This extension saves page source code and CSS styles including all modifications made by scripts and Opera.

This extension won't save page's JavaScripts. For correct saving create «text/phf» MIME type 
(Settings-Preferences-Advanced-Downloads-Add) with «htm» extension and «Save to disk» action.
